## Controls
Controls are configured so that mReader can be used one handed. 


| Button | Action |
|--|--| 
|DPAD/L-Stick/R-Stick| Scroll/Page |
|A/L1/R1| Enter |
|B/L2/R2| Back |
|X| Settings |
|Y/Start| TOC |
